# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** contato@tortoro.com.br

**Response SLA:** We will acknowledge your report within 72 hours and provide an initial assessment within 7 days.

**Scope:** This policy applies to the web application hosted at `agua-viva-landing.vercel.app`.

Please do **not** open a public GitHub issue for security vulnerabilities.

Thank you for helping keep this project secure.
